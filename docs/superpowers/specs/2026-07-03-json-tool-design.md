# JSON 工具设计规范

## 1. 概述

### 1.1 目标
在工具箱中新增 JSON 工具，提供 JSON 格式化、压缩、字符串转义、字符串反转义、校验等功能。

### 1.2 用户需求
- 格式化 JSON 字符串（美化，添加缩进）
- 压缩 JSON 字符串（去除空格和换行）
- 字符串转义（普通字符串转为 JSON 安全字符串）
- 字符串反转义（JSON 转义字符串还原为普通字符串）
- 校验 JSON 格式是否正确

### 1.3 成功标准
- 用户可以快速格式化和校验 JSON
- 支持拖拽导入文件
- 支持复制结果
- 操作简单直观

## 2. 架构设计

### 2.1 窗口架构
- **窗口类**：`JsonToolFrame.ts` - 位于 `src/main/frame/`
- **路由路径**：`/json-tool`
- **窗口尺寸**：900x600（与 Markdown 预览窗口一致）
- **窗口特性**：
  - 透明无边框窗口
  - 支持拖拽缩放
  - 初始位置屏幕正中心
  - 自定义标题栏（JSON 工具 + 版本号 + 最小化/关闭按钮）
  - 顶部渐变色条（品牌标识）

### 2.2 IPC 接口
| 接口 | 方向 | 说明 |
|------|------|------|
| `to-main-JsonTool:format` | 渲染→主 | 格式化 JSON |
| `to-main-JsonTool:compress` | 渲染→主 | 压缩 JSON |
| `to-main-JsonTool:escape` | 渲染→主 | 字符串转义 |
| `to-main-JsonTool:unescape` | 渲染→主 | 字符串反转义 |
| `to-main-JsonTool:validate` | 渲染→主 | 校验 JSON 格式 |
| `to-main-JsonTool:minimize` | 渲染→主 | 最小化窗口 |
| `to-main-JsonTool:close` | 渲染→主 | 关闭窗口 |

## 4. 功能实现

### 4.1 格式化
- 实现：`JSON.stringify(JSON.parse(input), null, 2)`
- 美化 JSON，添加 2 空格缩进
- 解析失败时显示错误信息

### 4.2 压缩
- 实现：`JSON.stringify(JSON.parse(input))`
- 去除所有空格和换行
- 解析失败时显示错误信息

### 4.3 转义
- 将普通字符串转为 JSON 安全字符串
- 替换特殊字符：`"` → `\"`，`\` → `\\`，换行 → `\n`，制表符 → `\t`
- 支持处理多行文本

### 4.4 反转义
- 将 JSON 转义字符串还原为普通字符串
- 反向替换：`\"` → `"`，`\\` → `\`，`\n` → 换行，`\t` → 制表符
- 支持处理多行文本

### 4.5 校验
- 尝试 `JSON.parse(input)`
- 成功：显示"✓ 校验通过"
- 失败：显示"✗ 校验失败"及错误信息

### 4.6 错误处理
- 所有功能在解析失败时，输出区显示红色错误信息
- 错误信息包含具体错误位置（行号、列号）
- 校验失败时，底部状态栏持续显示错误状态

## 5. 文件操作

### 5.1 拖拽导入
- 拖拽 `.json` 文件到输入区，自动读取文件内容

### 5.2 文件选择
- 点击输入区的"打开文件"按钮，选择 JSON 文件

### 5.3 保存结果
- 点击输出区的"保存文件"按钮，将结果保存为 `.json` 文件

### 5.4 文件编码
- 默认 UTF-8

## 6. 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+F | 格式化 |
| Ctrl+Shift+M | 压缩 |
| Ctrl+Shift+E | 转义 |
| Ctrl+Shift+U | 反转义 |
| Ctrl+Shift+V | 校验 |
| Ctrl+Shift+Delete | 清空 |
| Ctrl+C | 复制结果（输出区聚焦时） |

### 6.1 快捷键特点
- 全局快捷键，窗口聚焦时生效
- 快捷键显示在工具栏按钮的 tooltip 中
- 与现有应用快捷键不冲突

## 8. 集成点

### 8.1 工具箱集成
- 在 `Toolbox.vue` 的 `tools` 数组中添加 JSON 工具
- 添加对应的 `openTool` 逻辑

### 8.2 路由集成
- 在 `routes.ts` 中添加 `/json-tool` 路由
- 懒加载 JSON 工具组件

### 8.3 窗口工厂集成
- 在 `WindowFactory.ts` 中添加 `createJsonToolFrame` 方法
- 管理 JSON 工具窗口的生命周期

## 9. 依赖

- 无额外依赖，使用原生 JavaScript JSON API
- 复用现有窗口框架（BaseFrame）

## 10. 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 大文件解析性能 | 低 | 限制输入大小，显示警告 |
| 快捷键冲突 | 中 | 测试现有快捷键，确保不冲突 |
| 跨平台兼容性 | 低 | 使用标准 API，测试 Windows 和 macOS |
