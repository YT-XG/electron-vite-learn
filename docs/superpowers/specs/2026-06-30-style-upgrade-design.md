# 样式升级设计文档

## 概述

借鉴 Mineradio 项目的视觉效果，为 Electron-Vite-Learn 项目添加双主题支持和高级视觉效果。

## 设计决策

| 决策项 | 选择 | 原因 |
|--------|------|------|
| 主题模式 | 双主题（手动切换） | 用户明确选择 |
| CSS 方案 | 纯 CSS（移除 TailwindCSS） | 用户明确要求 |
| 玻璃质感 | 简洁毛玻璃 + 渐变叠加 | 效果好，复杂度适中 |
| 渐变文字 | 只用于主页标题 "妙妙屋" | 浅色主题下效果一般 |
| 骨架屏 | 只加到剪贴板历史列表 | 最常用列表，提升最明显 |
| 浮动动画 | 加到首页图标和文字 | 增加灵动感 |

## 实现范围

### 1. 双主题基础设施
- 在 `base.css` 中定义 CSS 变量（亮色/暗色）
- 通过 `data-theme` 属性切换
- 持久化到 localStorage

### 2. 玻璃质感
- **作用范围**: 侧边栏 + 弹窗（收藏弹窗、清空确认弹窗）
- **实现方式**: `backdrop-filter: blur()` + 半透明背景 + 渐变叠加层
- **深色主题**: 更强的玻璃效果，微妙的 RGB 色差

### 3. 渐变动画文字
- **作用范围**: 主页标题 "妙妙屋"
- **实现方式**: `background-clip: text` + 动画扫过效果
- **注意**: 浅色主题下效果一般，深色主题下更出彩

### 4. Shimmer 骨架屏
- **作用范围**: 剪贴板历史列表加载状态
- **实现方式**: CSS `@keyframes` shimmer 动画
- **替代**: 替换现有的 "加载中..." 文字

### 5. 浮动动画
- **作用范围**: 首页图标 (✦) 和欢迎文字
- **实现方式**: CSS `@keyframes` float 动画
- **效果**: 轻微上下浮动，增加灵动感
