# 搜索框 + Markdown 预览工具设计文档

> 创建日期：2026-07-02
> 状态：设计完成

## 概述

为「妙妙屋」应用新增两大功能：
1. **全局搜索框** — 快捷键呼出，快速启动工具、搜索剪贴板、打开应用/文件/网页
2. **Markdown 预览工具** — 多标签页实时分屏预览，支持拖入文件

---

## 一、全局搜索框

### 窗口规格

| 属性 | 值 |
|------|-----|
| 尺寸 | 600×400 |
| 位置 | 屏幕居中 |
| 样式 | 无边框、透明背景、毛玻璃效果 |
| 呼出快捷键 | `Ctrl+K`（Windows）/ `⌘K`（macOS） |
| 关闭方式 | Esc、点击窗口外部 |

### 搜索分类

| 类别 | 触发词 | 行为 |
|------|--------|------|
| 工具 | 直接输入 | 打开对应工具窗口 |
| 剪贴板 | `cb 关键词` | 搜索历史记录，回车复制 |
| 应用 | `app 名称` | 启动系统应用 |
| 文件 | `file 路径` | 用系统默认程序打开 |
| 网页 | `url 地址` 或完整 URL | 用浏览器打开 |

### 首字母搜索

支持拼音首字母模糊匹配，例如：
- `yl` → Markdown 预览（预览 = **Y**ù**L**ǎn）
- `dj` → 倒计时（倒计时 = **D**ào**J**ìshí）

### UI 交互

- 输入时实时搜索，展示分类结果
- 上下键选择，回车确认
- Esc 关闭

## 二、Markdown 预览工具

### 窗口规格

| 属性 | 值 |
|------|-----|
| 默认尺寸 | 900×600 |
| 最小尺寸 | 600×400 |
| 样式 | 无边框，支持拖拽标题栏 |
| 生命周期 | 按需创建，关闭后销毁 |

### 布局

```
┌──────────────────────────────────────────────────────────┐
│  📝 Markdown 预览                            [_][□][×] │
├──────────────────────────────────────────────────────────┤
│  [README.md] [笔记.md] [+]                        ×     │
├──────────────────────────────────────────────────────────┤
│                         │                                │
│    编辑区 (textarea)    │       预览区 (渲染结果)        │
│                         │                                │
│    # Hello World        │   # Hello World                │
│    这是 **粗体**        │   这是 粗体                    │
│                         │                                │
└──────────────────────────────────────────────────────────┘
```

### 多标签页功能

- `+` 按钮新建空白标签
- 支持拖入多个 `.md` 文件
- 每个标签独立编辑/预览
- 单标签关闭时隐藏，最后一个关闭时销毁窗口

### 技术选型

| 组件 | 方案 |
|------|------|
| Markdown 渲染 | `markdown-it` |
| 代码高亮 | `highlight.js`（按需加载） |

### 功能特性

- 左右分屏，支持拖拽调整比例
- 实时渲染 Markdown
- 支持拖入 `.md` 文件（读取内容）
- 支持保存（Ctrl+S）
- 代码高亮
- 主题跟随系统（亮/暗）

## 五、实现范围（第一期）

### 功能一：全局搜索框

1. 搜索框独立窗口（屏幕居中、毛玻璃效果）
2. 快捷键呼出/隐藏（`Ctrl+K`）
3. 首字母搜索支持（拼音首字母模糊匹配）
4. 搜索工具并打开
5. 搜索剪贴板历史并复制

### 功能二：Markdown 预览工具

1. Markdown 预览窗口（多标签页、分屏、实时预览）
2. 拖入文件支持
3. 基础保存功能

### 功能三：主界面工具箱

1. 侧边栏新增「工具箱」菜单
2. 工具卡片网格展示
3. 点击卡片打开对应工具
